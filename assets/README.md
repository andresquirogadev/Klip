# Klip Icon

Place your application icon here as `icon.png`

## Requirements

- **Format**: PNG (recommended) or ICO
- **Size**: 256x256 pixels minimum (for best quality)
- **Transparency**: Recommended for better visual integration

## File Name

The icon file must be named exactly: **`icon.png`**

## Path

```
d:\rEPOS\sql_snippet_dock\assets\icon.png
```

## What Happens If No Icon Is Present?

The application will use a default blue square as fallback.

## Where Is It Used?

- System tray icon
- All application windows (Login, Snippets, Settings, etc.)
- Compiled executable icon (when building with PyInstaller)

## Tips for Creating an Icon

1. Use a simple, recognizable design
2. Ensure it looks good at small sizes (16x16, 32x32)
3. Use contrasting colors for visibility
4. Avoid too much detail

You can create icons using:
- Adobe Illustrator / Photoshop
- GIMP (free)
- Inkscape (free)
- Online tools like Canva or Figma
